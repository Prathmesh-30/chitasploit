ChitraSploit Workflow

  • 1
    Client (React.js)

    A user submits the movie suggestion form.

  • 2
    Server (Node.js/Express.js)

    The server receives the request, validates the data, and prepares to save it.

  • 3
    Database (MongoDB)

    The validated data is written and stored as a new document in the collection.

  • 4
    Client (React.js)

    Receives a success response from the server and updates the UI with a confirmation.
